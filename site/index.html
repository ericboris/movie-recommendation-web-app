Project repository created and version control set up (e.g., Git).

Development environment configured with necessary tools and libraries.

  • Installed Node.js and npm via https://nodejs.org/en/ and verified by typing node -v and npm -v in the terminal.
  • Installed React and React-Router via npm install react react-dom react-router-dom and verified the installation by creating a new app by typing npx create-react-app new_app, cd new_app, and npm start in the terminal.

  • Installed MySQL Community Server via https://www.mysql.com/downloads/. Initial verification via mysql -V failed and I had to add a path variable via vim ~/.zshrc, export PATH=$PATH:/usr/local/mysql/bin/. My initial installation failed. I removed all

  • Installed Flask, Flask-SQLAlchemy, and Flask-CORS via pip3 install Flask Flask-SQLAlchemy Flask-CORS and verified the installation by creating a new directory mkdir flask_test_app, cd flask_test_app, and vim app.py. In app.py I wrote 

        from flask import Flask                                                                               
    from flask_sqlalchemy import SQLAlchemy                                                               
    from flask_cors import CORS                                                                           
    import pymysql                                                                                        

    app = Flask(__name__)                                                                                 
    CORS(app)                                                                                             

    app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql+pymysql://user_name:password@localhost/db_name'
    app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False                                                  
    db = SQLAlchemy(app)                                                                                  

    @app.route("/")                                                                                       
    def home():                                                                                           
        return "Hello World"                                                                              

    if __name__ == "__main__":                                                                            
        app.run(debug=True)
